Pusat Bantuan & FAQ

Temukan jawaban seputar penggunaan fitur-fitur admin DNR.412 Workshop

Dashboard

Halaman Dashboard menampilkan ringkasan data secara real-time, meliputi:
  • Total Antrian — jumlah total antrian yang terdaftar
  • Total Pelanggan — jumlah pelanggan yang sudah diregistrasi
  • Order Proses — jumlah order yang sedang dikerjakan
  • Order Selesai — jumlah order yang telah selesai
  • Antrian Terbaru — 5 antrian terakhir yang masuk
  • Order Terbaru — 5 order terakhir yang dibuat

Dashboard memuat data secara otomatis saat halaman dibuka. Jika data terasa tidak sinkron, cukup refresh halaman (tekan F5 atau Ctrl+R). Pastikan juga koneksi internet dan server backend dalam kondisi aktif.
Antrian

Antrian dibuat oleh pelanggan sendiri melalui halaman cek status / booking di sisi klien. Admin tidak perlu menambahkan antrian secara manual — fokus admin adalah memantau dan memperbarui status antrian yang masuk.

Di halaman Antrian, setiap baris data memiliki tombol Update Status. Klik tombol tersebut, lalu pilih status yang sesuai:
  • Pending — Antrian baru masuk, belum diproses
  • Proses — Motor sedang dalam pengerjaan
  • Selesai — Motor sudah selesai diservis
  • Batal — Antrian dibatalkan

Gunakan kolom pencarian di bagian atas tabel untuk mencari berdasarkan nama pelanggan atau nomor motor. Anda juga bisa menggunakan filter status untuk menyaring antrian berdasarkan statusnya. Klik tombol Reset untuk menghapus filter.
Pelanggan

Buka halaman Pelanggan lalu klik tombol + Tambah Pelanggan di pojok kanan atas. Isi form yang muncul dengan data pelanggan (nama, nomor telepon, email, dan alamat), lalu klik Simpan.

Di tabel pelanggan, kolom Aksi menyediakan dua tombol:
  • Edit — untuk mengubah data pelanggan
  • Hapus — untuk menghapus data (akan muncul konfirmasi)
Pastikan tidak menghapus pelanggan yang masih memiliki antrian atau order aktif.

Field yang wajib diisi (ditandai *) adalah Nama dan Nomor Telepon. Field Email dan Alamat bersifat opsional namun disarankan diisi untuk keperluan komunikasi.
Order Servis

Buka halaman Order Servis. Halaman ini langsung menampilkan form pengisian data pelanggan dan detail servis. Isi semua field yang diperlukan:
  • Nama Pelanggan dan Nomor Telepon
  • Nomor Polisi dan Jenis Motor
  • Deskripsi keluhan atau pekerjaan servis
  • Tanggal masuk, estimasi selesai, dan estimasi biaya
  • Status awal (default: Pending)
Setelah selesai, klik tombol Buat Order Servis.

Sidebar Order Terbaru di sebelah kanan menampilkan 10 order terakhir yang telah dibuat. Ini memudahkan admin untuk melihat riwayat order terkini tanpa harus menavigasi ke halaman lain. Klik tombol Refresh untuk memuat ulang daftarnya.

Akun & Login

Klik nama admin di pojok kanan navbar, lalu pilih Logout dari dropdown menu. Sistem akan menampilkan konfirmasi sebelum keluar untuk menghindari logout tidak sengaja.

Hal ini terjadi karena sesi login telah berakhir (token kadaluarsa). Sistem secara otomatis mengamankan halaman admin dengan mengecek autentikasi setiap kali halaman dimuat. Cukup login kembali dengan kredensial admin Anda.

Saat ini fitur lupa password belum tersedia di antarmuka ini. Jika lupa password, hubungi administrator sistem untuk melakukan reset password secara manual melalui database atau panel backend.
